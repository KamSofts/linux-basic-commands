### Files

```bash
#goto user's home folder /home/user
cd ~

#create a new files
touch school.txt
#or use file editor like nano, gedit, vim, etc., For example
nano school.txt

#rename files
mv school.txt school_days.txt

#move files
mkdir my_memories
mv school_days.txt my_memories/

#copy files
cd my_memories/
cp school_days.txt college_days.txt

#remove file(s)
rm -i collede_days.txt school_days.txt
```
