### Folders

```bash
#goto user's home folder /home/user
cd ~

#create a new folder
mkdir school

#rename folder
mv school school_days

#move folder
mkdir my_memories
mv school_days/ my_memories/

#copy folder
cp -r my_memories/school_days/ my_memories/college_days

#remove empty folder
rmdir my_memories/college_days/

#remove folder and its contents
rm -rf my_memories/
```
