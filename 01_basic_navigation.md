### Basic navigation commands
```bash
#Print Working Directory
pwd

#list directory contents
ls
#list one by one
ls -1
#list with file permission and details
ls -l
#list with hidden files and folders
ls -a
#get working folder details
pwd;ls -ld

#Change Directory
cd
#goto one level back
cd ..
#goto user's home folder (/home/user_name/)
cd ~
#goto root folder (/)
cd /
```
